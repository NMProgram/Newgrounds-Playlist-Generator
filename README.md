# About This Repository
This repository holds a console-based application that allows for the adding, removing and playing of all kinds of popular Newgrounds songs, as well as giving you the option to add ones of your own! Songs can also be filtered, updated and searched for using the terminal's Menu structure.

# Menu Structure
Legend:
- **Bold** = Action
- *Italic* = Menu

Main Menu:
  * *Alter Menu*
    + *Alter Song Menu*
      - **Add Song**
      - *Update Song Menu*
        * **Update ID**
        * **Update Name**
        * **Update Release Date**
        * **Update Genre**
        * **Update Level ID**
        * **Update Audio File**
      - **Delete Song**
    + *Alter Composer Menu*
      - **Add Composer**
      - *Update Composer Menu*
        * **Add Song**
        * **Update Song**
        * **Update Name**
        * **Update Join Date**
        * **Update Age**
        * **Update Description**
        * **Update Availability**
      - **Delete Composer**
  * *Search Menu*
    + *Search Song Menu*
    + *Search Composer Menu*
  * *Filter Menu*
    + *Filter Song Menu*
    + *Filter Composer Menu*
